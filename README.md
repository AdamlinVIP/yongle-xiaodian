> 仿《永乐大典》之体例，以**洪武正韵**为纲，采辑**公元2000年后**之新知新学。

## 🎯 编纂宗旨

本典依《永乐大典》"用韵以统字，用字以系事"之例，将2000年后涌现之科技、文化、商业新词，按《洪武正韵》七十六韵编排，汇为一典。
